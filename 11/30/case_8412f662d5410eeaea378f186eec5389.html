    事實及理由
一、原告主張：台北市○○區○○段二小段四一五之一、四一五之三地號土地（均由
    重測前五三六之一地號土地分割而來，下稱系爭土地）為原告子○○、卯○、丑
    ○○及癸○○四人分別共有，每人應有部分各為四分之一，惟被告辰○○所有門
    牌號碼為台北市○○路○段二十五巷六號之建物（如附圖甲１、甲２、甲３、甲
    ４部分）、被告壬○○所有門牌號碼為台北市○○路○段二十五巷八號之建物（
    如附圖乙部分）、被告丁○○、庚○○、辛○○、丙○○、戊○○、甲○○○、
    巳○○○、乙○○○及許淑真所共有門牌號碼為台北市○○路二十五巷十號（如
    附圖丙１、丙２部分）及十二號（如附圖丁１、丁２部分）之建物及被告寅○○
    所有門牌號碼為台北市○○路○段二十五巷十四號之建物（如附圖戊１、戊２部
    分）均分別無權占有系爭土地，原告自得本於所有權人地位請求被告拆屋還地。
    另被告無法律上原因而占有系爭土地，被告即受有相當於租金之利益致原告受有
    相當於租金之損害，故原告自得請求被告返還其所受利益並以自民國八十八年二
    月十日原告取得系爭土地所有權日起至被告返還系爭土地止，依土地法第九十七
    條第一項之規定，以申報地價年息百分之十計算之租金為計算被告所受利益之依
    據。並聲明如主文第一項至第四項所示，且聲明願供擔保請准宣告假執行。
二、被告則以：系爭土地於分割前係屬興隆段二小段四一五地號土地之部分，而興隆
    段二小段四一五地號土地於重測前之地號係興福段小段五三六之一，此土地原係
    原告所屬之祭祀公業陳忠記所有並以訴外人陳廷鏞為管理人，而訴外人陳廷鏞將
    原興福段小段五三六之一地號土地交予該祭祀公業派下員即訴外人陳高卻使用，
    而後訴外人陳高卻與訴外人閻江貴香訂有土地租賃契約而將該土地之使用收益權
    移轉予訴外人閻江貴香，訴外人閻江貴香復將系爭土地之使用收益權分別移轉交
    付予被告辰○○、訴外人陳清吉、許德旺（即被告丁○○、庚○○、辛○○、丙
    ○○、戊○○、甲○○○、巳○○○、乙○○○及許淑真之被繼承人）及被告寅
    ○○使用，而後訴外人陳清吉再將其權利移轉與被告壬○○，故被告就系爭土地
    均非無權占有。另被告寅○○所有位於台北市○○路○段二十五巷十四號之建物
    係由訴外人陳高卻合法興建，且訴外人許德旺欲於系爭土地改建房屋時，亦得訴
    外人陳廷鏞之同意，此益足證被告就系爭土地實有使用權利，故原告主張並不足
    採。縱認被告需對原告負不當得利之責，惟原告以申報地價年息百分之十作為計
    算依據，並無理由等語。並聲明：原告之訴及其假執行之聲請均駁回。
三、兩造不爭執之事實：
（一）系爭土地原為祭祀公業陳忠記所有，現所有權人為原告。
（二）系爭土地重測前之地號為台北市○○區○○段五三六之一號，八十六年七月之
      申報地價為每平方公尺新台幣（下同）二萬四千元，八十九年七月之申報地價
      為每平方公尺二萬五千六百元。
（三）被告辰○○為台北市○○路○段二五巷六號房屋之所有權人；被告壬○○為台
      北市○○路○段二五巷八號房屋之有權人；被告丁○○、庚○○、辛○○、丙
      ○○、戊○○、甲○○○、巳○○○、乙○○○及許淑真為訴外人許德旺之繼
      承人，為台北市○○路○段二五巷十號及十二號房屋之共有人；被告寅○○為
      台北市○○路○段二五巷十四號房屋之所有權人。上述房屋均未辦理第一次保
      存登記（見卷內第六八頁）。
四、本件原告主張被告無權占有，被告則辯稱有使用系爭土地之權利，經依民事訴訟
    法第二百七十一條之一準用同法第二百七十條之一第一項第三款之規定，整理並
    協議簡化本件之爭點為：（一）被告是否無權占有系爭土地？（二）原告按申報
    地價年息百分之十計算相當租金之不當得利是否過高？茲分述如下：
（一）被告無權占有系爭土地：
    (1)按「當事人主張有利於己之事實者，就其事實有舉證之責任。」，民事訴訟法
      第二百七十七條本文定有明文。查，被告既不否認原告為系爭土地之所有權人
      ，則被告即應就其有合法使用系爭土地之權源，為積極之舉證，應先敘明。
    (2)被告主張其有合法使用系爭土地之權源，固舉出祭祀公業陳忠記於五十五年七
      月五日出具與訴外人陳高卻之土地使用權同意書（見卷內第二四五頁），惟查
      ，依該同意書之文義內容觀之，原告之前手祭祀公業陳忠記僅同意訴外人陳高
      卻興建房屋，並未同意訴外人陳高卻以外之人興建系爭房屋。此外，被告亦無
      法積極舉證其所有之系爭房屋乃陳高卻所興建並輾轉移轉於渠等，是被告自不
      得僅憑該同意書即謂其等有合法使用系爭土地之權源。
    (3)被告雖另主張，依訴外人陳高卻與訴外人閻江貴香間之土地租賃契約（見卷內
      第一一八頁）；另被告寅○○與訴外人許德旺（即被告丁○○、庚○○、辛○
      ○、丙○○、戊○○、甲○○○、巳○○○、乙○○○、許淑真之被繼承人）
      分別與訴外人閻江貴香簽有杜賣證書（見卷內第二九至三０頁）；訴外人許德
      旺欲於系爭土地改建房屋時，亦得訴外人陳廷鏞（即祭祀公業陳忠記之管理人
      ）之同意（見卷內第三三頁），惟上述文書均屬私文書，經原告否認其真正後
      ，迄本院言詞辯論終結時止，被告並無法積極舉證證明上述文書之真正，則於
      無法證明上述私文書真正之情形下，本院自難以上述私文書為判斷被告是否有
      合法使用系爭土地之依據。更何況上述文書所載地號（訴外人陳高卻與訴外人
      閻江貴香間之土地租賃契約所載地號為五三六號及五三六之四號；被告寅○○
      與訴外人閻江貴香簽訂之杜賣證書上所載地號為五三八號）又與系爭土地重測
      前之地號（五三六之一號）不符，益足證被告上述私文書不足以憑為本院認定
      之依據。至被告所提出之使用執照及建造執照（見卷內第三一至三二頁）雖均
      載明業主為訴外人陳高卻，惟被告所有之系爭房屋是否為前述建造執照及使用
      執照所表彰之建物，被告迄至言詞辯論終結時止，亦無法之為積極之舉證，是
      本院自不得僅憑上開使用執照及建照執照，即行認定被告所有之房屋使用系爭
      土地為有合法之權源。
    (4)被告既無法積極舉證其等房屋有使用系爭土地之合法權源，則原告主張被告無
      權占有系爭土地即非無據，應予肯認。
    (5)又被告雖曾聲請本院命原告提出：祭祀公業陳忠記之派下員名冊。曾在系
      爭土地上設有地上權之陳古仁、陳銀燭之繼承系統表。訴外人陳高卻之繼承
      系統表。上述各項，經審俱與被告是否具有合法使用系爭土地之權源無直接相
      關，乃未依其請命原告提出，應併敘明。
（二）原告請求按申報地價年息百分之十計算相當租金之不當得利並未過高：
    (1)按「城市地方房屋之租金，以不超過土地及其建築物申報總價年息百分之十為
      限。」；「第九十七條第九十九條及第一百零一條之規定，於租用基地建築房
      屋均準用之。」，土地法第九十七條第一項及第一百零五條定有明文。次按「
      依不當得利之法則請求返還不當得利，以無法律上之原因而受利益，致他人受
      有損害為其要件，故其得請求返還之範圍，應以對方所受之利益為度，非以請
      求人所受損害若干為準，無權占有他人土地，可能獲得相當於租金之利益為社
      會通常之觀念，是被上訴人抗辯其占有系爭土地所得之利益，僅相當於法定最
      高限額租金之數額，尚屬可採。」，此亦有最高法院六十一年台上字第一六九
      五號判例意旨可供參照。
    (2)查系爭土地座落於景美地區，交通尚屬便捷，地點亦非偏僻，經審酌被告已長
      期享有無權使用系爭土地之不當得利，及原告僅請求自八十八年二月十日起算
      之相當於不當得利之租金等情，認原告依上述土地法之規定，請求按申報地價
      年息百分之十計算相當於租金之不當得利，尚屬允洽，茲分別計算原告所請求
      不當得利之數額如下：
      一、被告辰○○部分：
        １、自八十八年二月十日至八十九年六月卅日：24,000元x 115.97（平方公
            尺）x（1又365分之141年）x 10分之1 = 385,846元。
        ２、自八十九年七月一日起至九十二年七月卅一日：25,600元x115.97（平
            方公尺）x （3又12分之1年）x 10分之1 = 915,390  元。
        ３、自九十二年八月一日至交還土地日，按月：25,600 x 115.97（平方公
            尺）x 10分之1 x 12分之1 = 24,740元。
        ４、原告應有部分各四分之一，故被告應給付各原告之數額為：
          （１）（385,846 + 915,390）÷4 = 325,309 元。
          （２）自九十二年八月一日起至返還土地日止，按月6,185元（24,740÷4
                ）。
      二、被告壬○○部分：
        １、自八十八年二月十日至八十九年六月卅日：24,000 x 46.52（平方公尺
            ） x（1又365分之141年） x 10分之1 = 154,778 元。
        ２、自八十九年七月一日至九十二年七月卅一日：25,600 x 46.52（平方公
            尺） x（3又12分之1年） x 10分之1 = 367,198 元。
        ３、自八十九年八月一日至返還土地日止按月：25,600 x 46.52（平方公尺
            ）x 10分之1 x 12分之1 = 9,924元。
        ４、原告應有部分各四分之一，故被告應給付各原告之數額為
          （１）（154,778 + 367,198）÷4 = 130,494 元。
          （２）自九十二年八月一日起至返還土地日止，按月2,481元（9924 ÷4）
      三、被告丁○○、庚○○、辛○○、丙○○、戊○○、甲○○○、巳○○○、
          乙○○○、許淑真部分：
        １、自八十八年二月十日至八十九年六月卅日：24,000 x 89.25（平方公尺
            ） x（1又365分之141年） x 10分之1 = 296,946元。
        ２、自八十九年七月一日至九十二年七月卅一日：25,600 x 89.25（平方公
            尺） x（3又12分之1年） x 10分之1 = 704,480元。
        ３、自九十二年八月一日起至返還土地日止按月：25,600 x 89.25 x 10分
            之1 x 12分之1 = 19,040元。
        ４、原告應有部分各四分之一，故被告應給付各原告之數額為
          （１）（296,946 + 704,480）÷9 ÷4 = 27,817 元。
          （２）自九十二年八月一日起至返還土地，按月529元（19040 ÷9 ÷4）
      四、被告寅○○部分：
        １、自八十八年二月十日至八十九年六月卅日：24,000 x 51.54（平方公尺
            ） x（1又365分之141年） x 10分之1 = 171,480 元。
        ２、自八十九年七月一日至九十二年七月卅一日：25,600 x 51.54（平方公
            尺） x（3又12分之1年） x 10分之1 = 406,822 元。
        ３、自九十二年八月一日起至返還土地日止，按月：25,600 x 51.54（平方
            公尺）x 10分之1 x 12分之1 = 10,995元。
        ４、原告應有部分各四分之一，故被告應給付各原告之數額為
          （１）（171,480 + 406,822）÷4 = 144,576 元
          （２）自九十二年八月一日起至返還土地按月2749元（10995 ÷4）
五、綜上，原告主被告無權占有系爭土地，已堪認定，被告抗辯其有使用系爭土地之
    合法權源，尚不足取，從而，原告本於所有物返還請求權，請求被告拆屋還地，
    並依不當得利之法律關係，請求被告給付相當於租金之不當得利，並聲明如主文
    所示，為有理由，應予准許。又原告陳明願供擔保，聲請宣告假執行，經核尚無
    不合，乃分別酌定相當擔保金額准許之。
六、訴訟費用負擔之依據：民事訴訟法第三百八十五條第一項前段、第七十八條、第
    八十五條第一項。
