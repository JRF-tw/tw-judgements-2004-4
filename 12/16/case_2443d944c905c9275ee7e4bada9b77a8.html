【裁判字號】 92,訴,3451
【裁判日期】 931216
【裁判案由】 營利事業所得稅
【裁判全文】   
臺北高等行政法院判決　　　　　　　　　　　　　　九十二年度訴字第三四五一號
　　　　　　　　　　　　　　　.
　　原　　　告　廣用科技事業有限公司
　　代　表　人　甲○○
　　訴訟代理人　陳欽賢律師
　　被　　　告　財政部臺灣省北區國稅局
　　代　表　人　許虞哲（局長）
　　訴訟代理人　乙○○
右當事人間因營利事業所得稅事件，原告不服財政部中華民國九十二年四月二十三日
台財訴字第０九二０００五０四八號訴願決定，提起行政訴訟。本院判決如左：
　　主  　　文
原告之訴駁回。
訴訟費用由原告負擔。
　　事  　　實
一、事實概要：原告八十四年度營利事業所得稅結算申報，被告初查暫按申報核定其
    全年所得額為新台幣（下同）一二四、０九九元，嗣經財政部賦稅署（以下簡稱
    賦稅署）查得原告漏報未列成本之銷貨收入三、二八五、一００元及短報期末存
    貨六、四七一、三九一元，致短漏報所得七、一六一、二六二元，經被告重新核
    定全年所得額為七、二八五、三六一元，並就其漏報所得額部分補徵營利事業所
    得稅一、七九０、三一六元，復依所得稅法第一百十條第一項規定，就其所漏稅
    額一、七九０、三一六元科處一倍罰鍰一、七九０、三００元（計至百元止）；
    另原告當年度進貨計四、二一五、二二九元未依法取得憑證，被告依稅捐稽徵法
    第四十四條規定，按未取得憑證金額處百分之五罰鍰即二一０、七六一元，合計
    處罰鍰二、００一、０六一元。原告不服，申經復查結果，未獲准變更，提起訴
    願，經財政部於九十二年四月二十三日以台財訴字第０九二０００五０四八號訴
    願決定將原處分（復查決定）關於行為罰二一０、七六一元部分撤銷，其餘訴願
    則予以駁回。原告仍表不服，遂向本院提起行政訴訟。
二、兩造聲明：
  (一)原告聲明：
    1.訴願決定、復查決定及原處分均撤銷。
    2.訴訟費用由被告負擔。
　(二)被告聲明：
    1.駁回原告之訴。
    2.訴訟費用由原告負擔。
三、兩造爭點：原告是否有漏記系爭4M RAM MODULE產品進貨，而8M RAM MODULE產品
              則短報期末存貨之情形？
　(一)原告主張之理由：
    1.查原告接獲韓國DONG WON ELECTRONIC CO.,LTD 公司（以下簡稱韓商）訂購8M
      RAM MODULE 1000PCS，約定總價美金一三０、０００元、出貨期限為八十四年
      四月二十五日，並以空運出貨，原告遂向英基電子股份有限公司（以下簡稱英
      基公司）進貨，總價為六、七九四、九六一元（含稅）。嗣原告於八十四年四
      月二十四日預付六一四、一八一元貨款、四月二十五日出貨空運至韓國，待韓
      商確認收貨無誤後，再於四月二十八日付清餘款五、八五七、二一０元，分別
      於四月二十四日、四月二十八日取得英基公司開立之第00000000號、
      第0000000號統一發票各一張，發票上所載品名為8M RAM MODULE/70、
      數量1000 PCS，原告並隨即於進貨簿記載進貨品名「 G00052 1.C 」、單位數
      量1000 PCS、進貨金額六、四七一、三九一元。經查該批貨物於四月二十五日
      報關空運出口，出口報單上貨物名稱被報關行人員誤寫為「4M RAM MODULE 」
      ，但原告銷貨簿內仍記載銷貨品名為 G00052 1.C、單位數量 1000 PCS，與前
      開進貨簿所載相同，原告並開立第00000000號統一發票報稅。嗣原告
      發現出口報單品名書寫有誤，向財政部台北關稅局申請更正，然該局以前開出
      口報單已逾保存期限而銷毀為由，並未照准；惟原告已洽此貨物之承買人韓商
      證明前開出口到該公司之貨品為 8M RAM MODULE，而非 4M RAM MODULE，故本
      件進、銷貨品應為同批，而非如被告所稱之二批，是原處分及訴願決定顯有違
      誤。
    2.本件被告依據出口報單記載之 4M RAM MODULE 及同時期進貨發票上記載之 8M
       RAM MODULE兩項不同品名，逕核定原告外銷至韓國之1000PCS商品為兩批貨物
      ，並予補稅處罰，然此乃因出口報單上品名書寫有誤所致，系爭貨品實為同批
      貨物。蓋原告進貨時間為八十四年四月二十四日、出貨時間為四月二十五日，
      數量均為 1000PCS，足證進、銷貨品是同一批，否則不可能同時進出同數量之
      貨物二批；又原告當時進、銷貨簿均記載貨物品名為G00052 1.C、單位數量為
       1000PCS，而出口報單由報關行繕打難免有誤，故由韓商訂單及韓商開立之證
      明函可知原告交付與韓商之貨品為8M電腦記憶體而非4M電腦記憶體，是本件進
      、銷貨品既為同批貨物，則無銷貨三、二八五、一００元而未取得進貨發票之
      情事。況此筆銷貨原告既已列報當年度營業收入，原處分再以銷貨款三、二八
      五、一００元按同業利潤率百分之二十一計算漏報所得額六八九、八七一元，
      顯係重複課稅；另同時進貨之8M RAM MODULE 六、四七一、三九一元，既與外
      銷韓商者為同批貨物，自不能另計算銷貨收入，故原處分將此二項重複計算為
      漏報之營業收入，並補徵營利事業所得稅一、七九０三００元，於法不合。揆
      諸上開說明，被告不扣除任何營業成本，逕以進貨價六、四七一、三九一元全
      部視為原告之營業利潤，計為原告之所得課稅，顯有違誤。
  (二)被告答辯之理由：
    1.按「營利事業應保持足以正確計算其營利事業所得額之帳簿憑證及會計紀錄。
      」、「營利事業所得之計算，以其本年度收入總額減除各項成本費用、損失及
      稅捐後之純益額為所得額。」、「納稅義務人應於每年二月二十日起至三月底
      止，填具結算申報書，向該管稽徵機關，申報其上一年度內構成．．．營利事
      業收入總額之項目及數額，以及有關減免、扣除之事實，並應依其全年應納稅
      額減除暫繳稅額、尚未抵繳之扣繳稅額及可扣抵稅額，計算其應納之結算稅額
      ，於申報前自行繳納。」、「納稅義務人已依本法規定辦理結算申報，但對依
      本法規定應申報課稅之所得額有漏報或短報情事者，處以所漏稅額兩倍以下之
      罰鍰。」、「營利事業依法規定應給與他人憑證而未給與，應自他人取得憑證
      而未取得，或．．．，應就其未給與憑證、未取得憑證或．．，經查明認定之
      總額，處百分之五罰鍰」，分別為所得稅法第二十一條、第二十四條、第七十
      一條第一項、第一百十條第一項及稅捐稽徵法第四十四條所明定。次按「當事
      人主張事實須負舉證責任，倘其所提出之證據不足為其主張事實之證明，自不
      能認其主張之事實為真實。」，復為行政法院（現改制為最高行政法院）三十
      六年第一六號判例所明示。
    2.原告八十四年度營利事業所得稅結算申報，列報全年所得額一二四、０九九元
      ，經被告按書面審查核定全年所得額為一二四、０九九元，嗣經賦稅署查得八
      十四年四月二十五日帳載上載有一筆銷售IC產品至韓國，其出口報單上記載銷
      售品名為4M RAM MODULE，銷售數量1000PCS、銷售金額三、二八五、一００元
      ，惟原告帳載銷售該筆IC產品之進貨，其進貨發票所載產品品名卻為 8M RAM
      MODULE/70，銷售數量為1000PCS，進貨金額為六、四七一、三九一元，因前開
      銷售至國外之產品進貨與銷售品名不相符，又8M RAM MODULE/70產品亦未查得
      已銷售情事，致原告八十四年間所銷售 4M RAM MODULE之進貨未依法取得合法
      憑證二、五九五、二二九元及漏報所得額六八九、八七一元，另進貨8M RAM
       MODULE/70計六、四七一、三九一元亦未申報期末存貨，有行為時之發票、進
      銷簿及出口報單等資料可稽，乃據以核定全年所得額為七、二八五、三六一元
      ，並就其漏報所得額部分補徵營利事業所得稅一、七九０、三一六元。原告不
      服，復補提示韓商重申信函，稱其向原告進貨產品確是8M RAM MODULE/70，而
      非 4M RAM MODULE產品，惟因乏客觀事證可憑，仍不足為其主張事實之證明，
      又原告對原核定仍表不服，主張八十四年度出口一筆電腦組件品名 8M RAM
      MODULE/70、銷售數量為1000PCS，因公司職員繕打錯誤成4M RAM MODULE/70、
      銷售數量 1000PCS，又產品既為科技產品，生命週期較短且其存貨係採零存貨
      制度，此筆韓國訂單是接到訂單才下單進貨，故沒有預留存貨，資為爭議。惟
      查原告之帳載外銷交易資料有列報虧損三、六四六、九四四元，因此該筆交易
      ，如原告所言，係先有外銷韓國訂單後再向國內廠商英基公司下單進貨，由於
      該筆交易外銷價為已知，何以原告再以高出近一倍價格向英基公司進貨，顯不
      合理，另查IC產品4M RAM之單位售價為美金一三０元，而8M RAM之單價格高達
      美金二五七元，由於二者產品之單位售價差距頗大，故原告在出口報單上計算
      離岸價格時亦不可能計算錯誤；再查，依原告出口該IC產品之託運公司─欣洋
      通運有限公司所出具託運裝箱單據資料觀之，有關銷售該產品品名亦均載為4M
      RAM MODULE，而非原告所主張為銷售8M RAM產品；另原告主張八十四年度向英
      基公司所購買4M RAM產品（數量1000PCS、金額六、四七一、三九一元），業
      已取得進貨憑證入帳並於帳上沖轉4M RAM MODULE產品之銷貨成本，並無短報
      期末存貨云云，惟依相關資料查證結果，原告帳載期末存貨係採實地盤存制，
      除進貨、銷貨沖轉記錄與事實不符外，亦未查獲有銷售之事實及申報期末存貨
      ，故對於4M RAM MODULE產品誤列銷貨成本，而對於8M RAM MODULE/70產品短
      報期末存貨至明，故原核定予以補徵營利事業所得稅一、七九０、三一六元，
      認事用法並無不合。
    3.第查原告八十四年四月二十五日於帳載上計載一筆銷售IC產品至韓國，其出口
      報單上所載銷售品名為4M RAM MODULE，銷售數量為1000PCS，銷售金額三、二
      八五、一００元，惟原告帳載銷售該筆IC產品之進貨，其進貨發票卻載品名8M
      RAM MODULE/70，銷售數量1000PCS，進貨金額六、四七一、三九一元，經賦稅
      署查得，前開銷售至韓國之產品進貨與銷售品名不相符，又8M RAM MODULE/70
      產品亦未查得已銷售情事，致原告八十四年間所銷售4M RAM MODULE之進貨未
      依法取得合法憑證二、五九五、二二九元及漏報所得額六八九、八七一元至明
      ，又進貨產品8M RAM MODULE/70計六、四七一、三九一元亦未申報期末存貨，
      有行為時之發票、進貨簿及出口報單等資料可稽，違章事證明確，依前揭規定
      ，按所漏稅額一、七九０、三一六元處一倍罰鍰，即一、七九０、三００元（
      計至百元止）；至八十四年間銷售至韓國之IC產品 4M RAM MODULE，數量為
      1000PCS 計三、二八五、一００元部分，其進貨時亦未取得憑證二、五九五、
      二二九元，且向佳營電子股份有限公司（以下簡稱佳營公司）購買電子零件金
      額計一、六二０、０００元（未含稅），亦未依法取得佳營公司所開立發票，
      有行為時之發票、進銷簿及出口報單等資料可稽，違章事證明確，依稅捐稽徵
      法第四十四條規定，經就前開未取得憑證金額四、二一五、二二九元處百分之
      五罰鍰，即二一０、七六一元。原告不服，申經復查結果，未獲准變更，原告
      仍表不服，提起訴願，經財政部台財訴字第０九二０００五０四八號訴願決定
      ，就原處分關於行為罰二一０、七六一元部分撤銷，其餘漏稅罰部分訴願駁回
      ，原告仍未甘服，惟查原告八十四年間所銷售 4M RAM MODULE漏報所得額六八
      九、八七一元，及進貨產品8M RAM MODULE /70計六、四七一、三九一元亦未
      申報期末存貨，有行為時之發票、進貨簿及出口報單等資料可稽，違章事證依
      前述已然明確，且原告起訴意旨除執前詞爭執亦無新主張及新事證，自不能認
      其主張之事實為真實，是處漏稅罰鍰一、七九０、三００元部分並無違誤。至
      原告主張台北縣政府准予其自八十八年六月四日起歇業在案，致其未收到有關
      單位通知其提出利己事證之郵件云云，惟賦稅署於調查本件時，分別以八十八
      年七月二十七日臺稅稽發第0000000號及八月二十日臺稅稽發第八八０
      ０六七三號函請原告於文到五日內提出有利事證，有原告投資人林燦玉即原告
      代表人之配偶簽收之郵政掛號函件附原處分卷可按，是原告稱已歇業且未收到
      有關單位通知其提出利己事證一節，顯為推諉之詞，併此陳明。
　　理      由
一、本件起訴時被告之代表人林吉昌，嗣於本件訴訟程序進行中變更為許虞哲，茲由
    其聲明承受訴訟，核無不合，應予准許，合先敘明。
二、按「營利事業應保持足以正確計算其營利事業所得額之帳簿憑證及會計紀錄。」
    、「營利事業所得之計算，以其本年度收入總額減除各項成本費用、損失及稅捐
    後之純益額為所得額。」、「納稅義務人應於每年二月二十日起至三月底止，填
    具結算申報書，向該管稽徵機關，申報其上一年度內構成．．．營利事業收入總
    額之項目及數額，以及有關減免、扣除之事實，並應依其全年應納稅額減除暫繳
    稅額、尚未抵繳之扣繳稅額及可扣抵稅額，計算其應納之結算稅額，於申報前自
    行繳納。」，所得稅法第二十一條、第二十四條、第七十一條第一項分別定有明
    文。又按「納稅義務人已依本法規定辦理結算申報，但對依本法規定應申報課稅
    之所得額有漏報或短報情事者，處以所漏稅額兩倍以下之罰鍰。」，復為所得稅
    法第一百十條第一項所規定。
三、按當事人主張事實，須負舉證責任，倘其所提出之證據，不足為主張事實之證明
    ，自不能認其主張之事實為真實。行政法院三十九年判字第二號著有判例。本件
    原告八十四年度營利事業所得稅結算申報，被告初查暫按申報核定其全年所得額
    為一二四、０九九元，嗣經賦稅署查得原告漏報未列成本之銷貨收入三、二八五
    、一００元及短報期末存貨六、四七一、三九一元，致短漏報所得七、一六一、
    二六二元，被告乃重新核定予以補徵稅額並課處漏稅罰等情，有原告八十四年度
    營利事業所得稅結算申報書、出口報單、取貨單及發票等影本附於原處分卷可稽
    。原告雖主張係將進貨之「8M RAM MODULE」誤寫為「4M RAM MODULE」在出口報
    單上，而該外銷貨物業經課稅，被告再次補徵，顯係重複核課等語。惟按國際貿
    易上買賣契約之成立，以當事人就買賣標的物、價金等契約要素合意一致為必要
    ，故當事人對凡影響買賣標的物價金之因素，如品質、規格、數量、產地、包裝
    方式等，於買賣契約成立之初，當知之甚詳，遑論於出口前尤對出口貨物之內容
    當據以核對是否與買賣之標的物相符。第以原告既非初次出口貨物，衡情當對上
    開貿易及報關出關作業程序等節甚為熟稔，本無將所欲出口之買賣標的物裝載全
    然不同規格之貨物而有錯置混淆情形之可能；且系爭Ic 產品，4M RAM MODULE之
    單位售價為美金一三０元，而8M RAM MODULE 單價則高達美金二五七元，單位售
    價差距達一二七美元，自營利事業營業成本及獲利觀點觀之，原告斷無在出口報
    單上計算離岸價格時計算錯誤之理；參以託運之欣洋通運有限公司裝箱單據及取
    貨單暨發票（invoice）上均載明貨物品名為「4M RAM MODULE」字樣等情，足見
    所稱誤載貨物品名云云顯與國際貿易商業習慣有違，委無可取。又原告自承係先
    有外銷韓國訂單後始向國內廠商英基公司下單，依常情，原告既深知其外銷價格
    ，豈有反以高出近一倍價格向英基公司進貨之理，所為有違一般商業經營原理原
    則，本難採認；且原告既係出口業者，果所稱系爭貨物係誤載品名為真，則其應
    依行為時出口報單申報錯誤審核更正作業要點相關規定，於八十四年間出口後四
    個月內申請更正，但原告卻遲至九十一年十二月十日（申請書日期登載為九十一
    年十一月四日）始申請更正，自難信為真正；況徵之一般國際貿易商業習慣，苟
    本件確有記載出貨貨名錯誤之情形，按常情當有進、出口雙方就該事商討解決之
    往來文件資料，況原告身為出貨人，焉有得悉出貨誤載情形，卻置之不為更正之
    理，況如其確係出貨高價之8M RAM MODULE 產品，豈有坐任虧損並另須擔負違約
    賠償責任之理；加以原告於申請復查時，業經提出韓商九十一年二月二十四日補
    充說明資為其出口產品係8M RAM MODULE之佐證，可見其於被告復查決定前即已
    知悉其出口登載記錄，所稱因疏忽而未注意到有誤載情形，迄收受復查決定後方
    發現云云，非惟與上開事實相悖，殊無足採，且其迄未提出韓商進口報單之相關
    資料及證據以實其說，徒以韓商要約即 order為據，乏其 acceptance及契約（
    contract）等帳證，復空言當年度之虧損係因其違約所致，卻未提出積極證據以
    實其說，徵之前開判例，自難謂其已盡舉證責任，所言殊難採信，是原告於系爭
    八十四年間所銷售4M RAM MODULE之進貨有未依法取得合法憑證二、五九五、二
    二九元之事實，堪以認定。再原告主張其於系爭年度向英基公司購買之4M RAM
    MODULE產品（數量1000PCS、金額六、四七一、三九一元）業經取得進貨憑證入
    帳並於帳上沖轉4M RAM MODULE產品之銷貨成本，故無短報期末存貨一節，經查
    原告系爭年度係採實地盤存制，依卷附發票、進貨簿及出口報單等帳載資料，4M
    RAM MODULE產品雖有銷貨收入六八九、八七一元之記載，惟漏報所得額，而前所
    進貨之存貨即8M RAM MODULE/70記憶體計六、四七一、三九一元則無銷貨及申報
    期末存貨之情形，是所稱進貨、銷貨沖轉記錄顯與事實不符，且4M RAM MODULE
    漏記進貨部分，被告僅核定短報所得額，未核定短報銷貨收入，並無原告所稱重
    複課稅情形，原告所稱殊有誤解。故原告確有4M RAM MODULE產品誤列銷貨成本
    、8M RAM MODULE/70產品短報期末存貨之事實，至堪認定。則被告以上開銷售至
    韓國之產品進貨發票與銷售品名不相符，且8M RAM MODULE/70產品查無已銷售情
    事，遂認原告於八十四年間所銷售之4M RAM MODULE 有未依法取得合法憑證二、
    五九五、二二九元及漏報所得額六八九、八七一元，而進貨之8M RAM MODULE/70
    計六、四七一、三九一元有未申報期末存貨之事實，即非無憑。從而被告以原告
    短漏報所得額，而予以補徵稅額並課處漏稅罰，所為處分，揆諸首揭法條規定，
    並無違誤，訴願決定予以維持，亦無不合，原告之訴難認有理由，自應予以駁回
    。
據上論結，本件原告之訴為無理由，依行政訴訟法第九十八條第三項前段，判決如主
文。
